Инициализация проекта
Перед началом работы инициализируем проект. Для этого откроем терминал в Visual Studio Code через меню View > Terminal и выполним следующую команду:
npm init -y
package.json — это файл конфигурации проекта, который применяется npm для управления настройками проекта и пакетами приложения. Пока можем оставить настройки в этом файле по умолчанию.
index.js - Напишем в index.js простой код для тестирования сборки:
Установим сборщик Rollup с помощью следующей команды: npm install --global rollup

После этого создадим файл конфигурации Rollup rollup.config.mjs (именно .mjs): 
      export default {
        input: './index.js',
        output: {
          file: './build/bundle.js',
          format: 'cjs'
        }
      };

Описание настроек:
input – входной файл, указываем путь к основному JS-файлу
output:
file – выходной файл, куда будут собраны все файлы проекта
format – формат сборки, по умолчанию это "cjs" (CommonJS).
Точка и слэш ./ в пути к файлу означают текущую дерикторию с файлом (в данном случае директорию файла rollup.config.mjs, корневую директорию проекта).
Более подробное описание настроек можно найти в документации Rollup.

Сборка проекта Чтобы собрать проект с помощью Rollup, достаточно запустить в терминале команду сборки с текущей конфигурацией: rollup -c
